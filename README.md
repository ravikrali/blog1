# R2 Data Vision (Premium Static Site)

A premium, minimal dark-mode technical blog with:
- 10 posts (each includes a Mermaid diagram)
- Client-side full-text search (Fuse.js)
- RSS-based AI news aggregator (rss2json)
- About page optimized for senior leadership roles (no company names)

## Deploy
- GitHub Pages: upload folder contents to a repo and enable Pages.
- Netlify / Vercel: drag-and-drop or connect repo.

## Customize
- Edit `styles.css` for theme
- Edit `app.js` to add RSS feeds or tune search behavior
- Add posts under `/posts/` and update `search-index.json`
